# Vision2Share – CREA 3.0

## Descripción
Vision2Share es un dispositivo de asistencia diseñado para personas con discapacidad visual.
El sistema utiliza sensores y retroalimentación táctil/sonora para ayudar en la detección de
obstáculos y mejorar la movilidad cotidiana.

El proyecto fue desarrollado como parte del **Concurso CREA 3.0**, combinando ingeniería,
diseño y enfoque social.

---

## Problema que aborda
Las personas con discapacidad visual enfrentan:
- Dificultades para detectar obstáculos cercanos
- Dependencia de herramientas limitadas o costosas
- Falta de soluciones accesibles y personalizables

---

## Solución propuesta
Vision2Share ofrece:
- Detección de obstáculos mediante sensores
- Retroalimentación inmediata al usuario
- Diseño portable y adaptable
- Enfoque en accesibilidad y bajo costo

---

## Tecnologías utilizadas
- **Hardware:** Arduino Nano, sensores de distancia, motores de vibración
- **Software:** Arduino IDE
- **Diseño:** Tinkercad, impresión 3D
- **Electrónica:** Soldadura, cableado y ensamblaje

---
